Baseline & Evaluation Plan 8 / 11

Baseline Model

Majority Class Predictor — Always predict "On Time"

Expected accuracy: ~57.7%

This is not useful operationally — it never catches a delay. Every real model must beat this.

Classification Metrics

Primary: F1 Score

Why? Classes are imbalanced (57.7% vs 42.3%). Accuracy is misleading. F1 balances precision and recall.

Also: ROC-AUC for ranking prediction quality.

Regression Metrics

MAE — Average error in minutes, directly interpretable for passengers.

RMSE — Penalizes large delay mistakes. A model with MAE 8 min but RMSE 25 min is unreliable on severe delays.

Evaluation Protocol

Time-based split: Chronological, no random shuffle — prevents temporal leakage.

Ablation study: Same algorithm with vs without weather features → directly answers RQ2.
